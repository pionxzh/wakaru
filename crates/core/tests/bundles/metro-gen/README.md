# Metro generated fixtures

These bundles are generated by Metro 0.87.0 with `transformer.globalPrefix`
set to `metro$`. The source covers static dependencies plus the dependency-map
forms emitted for dynamic import, prefetch, maybe-sync import, and
`require.resolveWeak()`.

The generator replaces its absolute working directory with
`<METRO_FIXTURE_ROOT>` in debug-only module-name strings, keeping the committed
artifacts reproducible across checkouts.

Regenerate both checked-in artifacts with:

```sh
npm install
npm run generate
```
